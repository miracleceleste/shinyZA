## Special Characters Note

**Flabébé** - Remove the accent, use: `flabebe.jpg`

## File Format

All images should be:
- JPG or PNG format
- Named exactly as shown (all lowercase)
- Placed in the `Images` folder
- Recommended size: 150-300px

## Don't Forget!

Also add: `lumiose-map.jpg` - The Lumiose City map for the zone selector

## Batch Naming Tip

If you're renaming files in bulk:
1. Download all shiny sprites
2. Use a bulk rename tool
3. Convert all to lowercase
4. Replace spaces with nothing
5. Remove special characters
6. Add .jpg extension

## Missing Images

The tracker will still work if images are missing - they just won't display. Add images gradually as you hunt!
